🚀 Google Cloud Run Setup Guide

1 Go to Cloud Run

Open Cloud Run

Make sure you're in project: My First Project

2 Create Service

Click: CREATE SERVICE

Fill in:

  • Service name: openclaw-wholescaleos
  • Region: us-central1 (Iowa)

3 Container Settings

Under "Container" section:

  1. Select: Continuously deploy new revisions from a source repository
  2. Click: SET UP WITH CLOUD BUILD
  3. Select: GitHub (Cloud Build GitHub App)
  4. Click: CONNECT

Note: If you don't want to connect GitHub, we can use Cloud Shell instead.

4 Repository Connection

After connecting GitHub:

  1. Select repository: [I'll create one for you]
  2. Branch: main
  3. Build type: Dockerfile
  4. Dockerfile location: /Dockerfile

5 Service Configuration

Scroll down and set:

  • Authentication: ☑ Allow unauthenticated invocations
  • Container port: 8080
  • Memory allocated: 512 MiB
  • CPU: 1
  • Minimum instances: 0
  • Maximum instances: 1

6 Environment Variables

Click: ADD VARIABLE

  • Name: RUN_MODE
  • Value: cloud

Click: ADD VARIABLE again

  • Name: OPENCLAW_WORKSPACE
  • Value: /app/workspace

7 Create & Deploy

Click: CREATE

Wait 5-10 minutes for deployment

8 Get Your URL

After deployment, you'll get a URL like:

https://openclaw-wholescaleos-xxxxxxxx-xx.a.run.app

Bookmark this URL - it's your 24/7 OpenClaw!
